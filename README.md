# Handwritten Character Recognition

## Objective
The objective of this project is to identify handwritten digits using Deep Learning and Convolutional Neural Networks (CNN).

## Dataset
MNIST Dataset

- 60,000 training images
- 10,000 testing images
- Image size: 28x28 pixels
- Digits: 0 to 9

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Model Architecture
- Conv2D
- MaxPooling2D
- Flatten
- Dense Layers
- Softmax Output Layer

## Features
- Image preprocessing
- CNN-based classification
- Model training and evaluation
- Handwritten digit prediction
- High accuracy recognition

## Results
The model achieved high accuracy on the MNIST test dataset.
